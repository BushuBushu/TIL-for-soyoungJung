### Bottom-up

1. dynamic table을 2차원 리스트로 둔다.

2. d[i][j]는 출발지(0,0)로부터 소요되는 에너지를 의미한다.

3. d의 0행과 0열을 모두 초기화한다.(값을 누적해서)

4. d[1][1]부터 값을 구한다. (12시방향, 9시 방향 중 작은 값을 누적)

5. d[n][n]의 값을 구한다. (=정답)
